## Branch Naming Convention

꼭 `main` 브랜치에서 생성해주세요!

```bash
    git branch feature/{이름}
```

- 생성 후 해당 브랜치로 바로 이동합니다.

```bash
    git switch feature/{이름}
```

## Install package.json

- Node.js 프로젝트에 package.json 파일 기반 의존성 패키지를 설치합니다.

```bash
    npm install # 혹은 npm i
```
